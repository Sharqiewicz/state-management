 # state-management-sharqiewicz

  Claude Code skill for React state management — Zustand, React Query, React Context, XState.

  ## Install

  Global (available in all projects):
  ```bash
  npx skills add Sharqiewicz/state-management --global
  ```

  Per-project:
  ```bash
  npx skills add Sharqiewicz/state-management
  ```

  ## What it covers

  - **Zustand** — store patterns, slices, devtools
  - **React Query** — server state, caching, mutations, optimistic updates
  - **React Context** — when to use it vs. Zustand
  - **XState** — state machines for complex UI logic
  - Common mistakes and review checklist

  ## Activation

  The skill activates automatically when you mention: `Zustand`, `React Query`, `TanStack Query`, `XState`, `useQuery`,
   `useMutation`, `server state`, `invalidateQueries`, `staleTime`, etc.
